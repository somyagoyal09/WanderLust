# 🏡 WanderLust

A full-stack Airbnb-inspired web application that allows users to explore, create, edit, review, and manage travel accommodations. Built with the MERN backend stack using secure authentication, image uploads, and role-based authorization.

---

## ✨ Features

- 🔐 User Authentication (Signup, Login & Logout)
- 🏠 Create, Edit & Delete Listings
- 🖼️ Image Upload Support
- ⭐ Add & Delete Reviews with Ratings
- 👤 Authorization (Only Owners can Edit/Delete)
- 💬 Flash Messages & Validation
- 📱 Responsive UI
- 🗄️ MongoDB Database Integration
- 🔒 Session & Cookie Management

## 🚀 Installation

```bash
git clone https://github.com/somyagoyal09/wanderlust.git

cd wanderlust

npm install

npm start
```

---

## 💻 Usage

1. Create an account or log in.
2. Browse available listings.
3. Create your own property listing.
4. Upload listing images.
5. Edit or delete your own listings.
6. Leave reviews and ratings on listings.
